# WorkBuddy操作指南

本文档说明如何将 WorkBuddy 任务关联到本文件夹。

## 操作步骤

1. 打开 WorkBuddy，并进入目标任务。
2. 在任务说明或备注中填写本地文件夹路径：

```text
D:\项目文件夹
```

3. 如果任务对应具体项目，建议在 `项目文件` 中新建项目专属文件夹，例如：

```text
D:\项目文件夹\项目文件\P001_项目名称
```

4. 在 WorkBuddy 任务中记录对应项目路径，方便后续快速定位资料。

## 使用建议

- 一个 WorkBuddy 任务尽量对应一个项目文件夹。
- 任务资料统一存放在 `项目文件` 内。
- 说明类文档统一存放在 `项目使用说明书` 内。
- 任务完成后，可在 WorkBuddy 中备注最终文件路径和版本号。
